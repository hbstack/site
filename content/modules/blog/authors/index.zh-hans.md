{{< bs/config-toggle >}}

## Authors Parameters

详细地介绍作者。

| Parameter     |  Type  | Description                                                                |
| ------------- | :----: | -------------------------------------------------------------------------- |
| `title`       | string | 作者名称。                                                                 |
| `description` | string | 作者描述。                                                                 |
| `email_hash`  | string | 邮箱的 `md5` 哈希值，用于生成 Gravatar 头像，也可以使用下面的 `images`。 |
| `images`      | array  | 第一张图片将作为头像。                                                     |
| `socials`     | object | 另请参阅[社交链接]({{< ref "modules/components/socials" >}})。                   |
