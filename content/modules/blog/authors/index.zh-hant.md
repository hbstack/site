## Authors Parameters

詳細地介紹作者。

| Parameter     |  Type  | Description                                                                |
| ------------- | :----: | -------------------------------------------------------------------------- |
| `title`       | string | 作者名稱。                                                                 |
| `description` | string | 作者描述。                                                                 |
| `email_hash`  | string | 郵箱的 `md5` 哈希值，用於生成 Gravatar 頭像，也可以使用下面的 `images`。 |
| `images`      | array  | 第一張圖片將作爲頭像。                                                     |
| `socials`     | object | 另請參閱[社交鏈接]({{< ref "modules/components/socials" >}})。                   |
