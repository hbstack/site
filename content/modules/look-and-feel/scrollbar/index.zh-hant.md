該模塊修改瀏覽器默認的滾動條樣式。

<!--more-->

{{% hb-module scrollbar %}}

## 站點參數

| Parameter   |  Type  |  Default  | Description                      |
| ----------- | :----: | :-------: | -------------------------------- |
| `height`    | string |  `12px`   | 滾動條高度。                     |
| `width`     | string |  `12px`   | 滾動條寬度。                     |
| `track_bg`  | string | `#f8f9fa` | 滾動條軌道背景顏色。             |
| `thumb_bg`  | string | `#909294` | 滾動條滑塊背景顏色。             |
| `corner_bg` | string | `#909294` | 滾動條交匯處背景顏色，如右下角。 |
