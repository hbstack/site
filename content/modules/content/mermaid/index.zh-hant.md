Mermaid 是一個基於 JavaScript 的圖表工具，使用 Markdown 啓發的文本定義和一個渲染器來創建和修改複雜的圖表。

<!--more-->

{{% hb-module mermaid %}}

## 用法

### Mermaid 代碼塊

````markdown
```mermaid
DIAGRAM SYNTAX
```
````

### Mermaid 短代碼

```markdown
{{</* mermaid */>}}
DIAGRAM SYNTAX
{{</* /mermaid */>}}
```

## 例子

詳情請參閱 [Mermaid Docs](https://mermaid.js.org/)。

### 流程圖

```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```
